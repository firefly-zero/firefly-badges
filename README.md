# firefly-badges

System app for [Firefly Zero](https://fireflyzero.com/) for viewing badges (achievements) for other apps.

## Installation

1. [Install firefly-cli](https://github.com/firefly-zero/firefly-cli?tab=readme-ov-file#installation)
1. Run `firefly_cli import sys.badges`

After it is installed, select "achievements" from the app menu in launcher and it will launch the achievements viewer.
